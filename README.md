# ReactiveX JS

A sample currency converter application using ReactJS and ReactiveX JS.


######Setup Guide######

Setup commands:
```
npm install;
```


######Running the Application######

::For development
```
npm run dev;
```
:: For production
```
npm run prod;
```
:: Then open index.html in your favorite web browser



Tech:
- ReactJS
- Reactive JS
- EcmaScript2015
- NodeJS
- Webpack
- REST API
